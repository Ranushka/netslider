netslider
=========

simple full width slider

yet file size .min -> 2kb


to use this slider

* add jQury
* link plugin


HTML markup is
--------------
```html
<div id="sample" >
    <ul>
        <li><img src="img/1.jpg" alt=""></li>
        <li><img src="img/2.jpg" alt=""></li>
        <li><img src="img/3.jpg" alt=""></li>
        <li><img src="img/4.jpg" alt=""></li>
        <li><img src="img/5.jpg" alt=""></li>
    </ul>
</div>
```

JAVASCRIPT is
-------------
```html
<script>
    $(function () {
        $( "#sample" ).netSlider();
    })
</script>
```
